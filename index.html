Livro digital do curso PAG014903

Fundamentos de Programação em Python

Guia prático e offline para conectar Python ao geoprocessamento.

Por que este material existe?

Esta coletânea acompanha a disciplina PAG014903 — Programação Aplicada ao Geoprocessamento e foi projetada para funcionar em qualquer dispositivo, mesmo sem internet. Use-a como trilha principal de estudos e como referência rápida em sala de aula ou em campo.

Versão 0.0.1 · Atualização 2025-2

Bem-vinda(o) à disciplina!

Conecte os fundamentos de Python às demandas do geoprocessamento com exemplos comentados, exercícios guiados e desafios reais. Leia no seu ritmo, retome conceitos quando quiser e marque os trechos essenciais.

  • Objetivos rápidos

    Mostrar como cada conceito de programação se conecta a aplicações espaciais e a dados reais do IFSC.

  • Como usar

    Mantenha todos os arquivos juntos para navegar offline. O layout responde bem em celulares, tablets e desktops.

  • Evolução contínua

    Este material é vivo e incorpora rapidamente sugestões da turma. Volte com frequência para ver novidades.

Sobre este livro

Este livro digital foi concebido como material de apoio para a disciplina PAG014903 — Programação Aplicada ao Geoprocessamento, ofertada no Curso Técnico em Geoprocessamento do IFSC (Câmpus Florianópolis – Mauro Ramos). Ele busca equilibrar rigor técnico e linguagem acessível.

O conteúdo é atualizado continuamente com base nas necessidades das turmas e em feedbacks recebidos. Recomenda-se conferir regularmente a versão mais recente.
