Capítulo 7: Conectando com o Brasil - A API de Serviços do IBGE

Olá! Até agora, trabalhamos com dados que nós mesmos criamos ou que estavam em arquivos locais (como os arquivos CSV). Mas e se precisássemos de dados atualizados sobre a população de todos os municípios de Santa Catarina? Ou o formato vetorial (a "malha") de uma cidade específica? Manter esses dados atualizados manualmente seria impossível.

É aqui que entram as APIs (Interfaces de Programação de Aplicativos). Pense em uma API como um "portal de atendimento" online para um grande banco de dados. Em vez de baixar um gigantesco arquivo com todos os dados do Brasil, nós, através do nosso código Python, fazemos uma "pergunta" (uma requisição) a esse portal, pedindo apenas os dados que nos interessam. O portal (a API) então nos devolve uma "resposta" estruturada, geralmente no formato JSON, contendo exatamente a informação que pedimos.

A API de Serviços do IBGE é um desses portais, mantido pelo Instituto Brasileiro de Geografia e Estatística. Ela nos dá acesso programático a uma vasta gama de dados públicos, oficiais e atualizados sobre o Brasil, incluindo:

  • Malhas Territoriais: Os limites vetoriais de estados, municípios, distritos, etc.
  • Localidades: Listas de todos os estados, municípios, seus nomes e códigos.
  • Dados Estatísticos: Informações de pesquisas como o Censo (população, domicílios, etc.).
  • Rede Geodésica: A localização e os dados das estações GNSS de referência no Brasil.

Neste capítulo, vamos aprender a "conversar" com a API do IBGE para buscar esses dados e integrá-los em nossos projetos.

2. A Ferramenta Essencial: A Biblioteca requests

Para fazer requisições a uma API (ou a qualquer outro recurso na internet), usamos uma biblioteca Python chamada requests. Ela simplifica enormemente o processo de comunicação via HTTP, o protocolo padrão da web.

Passo 1: Instalar a biblioteca requests

Se você ainda não a tem, abra o terminal do seu VS Code e digite:

pip install requests

Passo 2: Sua primeira requisição

Vamos fazer um teste simples: buscar a lista de todos os estados do Brasil. A URL (o "endereço" do serviço que queremos acessar) para isso é https://servicodados.ibge.gov.br/api/v1/localidades/estados.

import requests

# 1. Definir a URL do serviço (endpoint)
url_estados = "https://servicodados.ibge.gov.br/api/v1/localidades/estados"

print("Fazendo a requisição para a API do IBGE...")
try:
    # 2. Fazer a requisição GET
    response = requests.get(url_estados)
    response.raise_for_status()  # Lança um erro se a requisição falhou (ex: erro 404, 500)

    # 3. Converter a resposta para o formato JSON
    dados = response.json()

    # 4. Explorar os dados recebidos
    print(f"A requisição foi um sucesso! Foram encontrados {len(dados)} estados.")

    # Vamos imprimir o nome e a sigla dos 5 primeiros estados
    print("\nOs 5 primeiros estados na lista são:")
    for estado in dados[:5]:
        print(f"- {estado['nome']} ({estado['sigla']})")

except requests.exceptions.RequestException as e:
    print(f"Ocorreu um erro na requisição: {e}")

O que este código faz?

  • requests.get(url): envia uma requisição HTTP do tipo GET para a URL especificada. É como digitar o endereço no seu navegador e apertar Enter.
  • response.raise_for_status(): linha de segurança que interrompe o programa caso o servidor retorne um código de erro.
  • response.json(): converte o texto da resposta (em JSON) diretamente para uma estrutura Python pronta para uso.

3. Explorando os Serviços (Endpoints) da API

Agora que sabemos como fazer uma requisição, vamos explorar os serviços mais úteis para o geoprocessamento.

3.1. Nomes e Códigos de Cidades (Localidades)

Um passo fundamental em muitos fluxos de trabalho é obter o código oficial de um município, pois ele é usado para buscar outros dados, como sua malha ou estatísticas.

import requests

# URL para buscar os municípios de um estado (UF). 42 é o código de Santa Catarina.
# Para outros estados, troque o código. Ex: SP = 35, RJ = 33.
uf_code = "42"  # Código de Santa Catarina
url_municipios_sc = f"https://servicodados.ibge.gov.br/api/v1/localidades/estados/{uf_code}/municipios"

response = requests.get(url_municipios_sc)
municipios = response.json()

print(f"Encontrados {len(municipios)} municípios em Santa Catarina.")

# Vamos encontrar o código de Florianópolis
for municipio in municipios:
    if municipio['nome'] == "Florianópolis":
        codigo_floripa = municipio['id']
        print(f"\nO código IBGE de Florianópolis é: {codigo_floripa}")
        break

3.2. Malhas Territoriais (Limites Vetoriais)

A API pode fornecer o limite vetorial de qualquer localidade em formatos como GeoJSON, que podemos usar diretamente em bibliotecas como o Folium.

O formato da URL é https://servicodados.ibge.gov.br/api/v3/malhas/municipios/{codigo_municipio}.

import requests
import folium

# Usando o código de Florianópolis que encontramos: 4205407
codigo_municipio = "4205407"
url_malha = f"https://servicodados.ibge.gov.br/api/v3/malhas/municipios/{codigo_municipio}"

# Parâmetros para pedir o formato GeoJSON e a resolução máxima
params = {'formato': 'application/vnd.geo+json', 'resolucao': '5'}

print("Buscando a malha municipal de Florianópolis...")
response = requests.get(url_malha, params=params)
malha_geojson = response.json()

print("Malha recebida! Criando o mapa...")

# --- Criando um mapa com a malha ---
# Vamos centralizar o mapa na primeira coordenada da malha
coords_exemplo = malha_geojson['features'][0]['geometry']['coordinates'][0][0]
mapa_malha = folium.Map(location=[coords_exemplo[1], coords_exemplo[0]], zoom_start=10)

# Adicionando a malha GeoJSON ao mapa
folium.GeoJson(
    malha_geojson,
    name='Limite Municipal'
).add_to(mapa_malha)

folium.LayerControl().add_to(mapa_malha)
mapa_malha.save("mapa_com_malha_floripa.html")
print("Mapa 'mapa_com_malha_floripa.html' salvo com sucesso!")

3.3. Dados Estatísticos (Agregados da Pesquisa)

A API do IBGE também dá acesso a dados de pesquisas, como o Censo, por meio do sistema SIDRA. Para buscar essas informações precisamos conhecer o código da tabela, das variáveis e dos níveis geográficos.

Vamos buscar um dado econômico relevante: o PIB per capita dos municípios.

import requests
import pandas as pd

# Código de Santa Catarina
uf_code = "42"
# Tabela de 'Produto Interno Bruto dos Municípios'
tabela = "5938"
# Variável 'Produto Interno Bruto per capita, a preços correntes'
variavel = "37"

# A URL é montada para o SIDRA
url_pib = f"https://servicodados.ibge.gov.br/api/v3/agregados/{tabela}/periodos/2021/variaveis/{variavel}?localidades=N6[N3[{uf_code}]]&classificacao=2[all]"

response = requests.get(url_pib)
dados_pib = response.json()

# --- Processando os resultados do SIDRA ---
# A resposta do SIDRA é uma lista; vamos pegar o primeiro item
dados_variavel = dados_pib[0]

# Os resultados vêm em um dicionário onde a chave é o código do município
resultados = dados_variavel['resultados'][0]['series'][0]['serie']

# Vamos transformar isso em uma tabela mais amigável com Pandas
lista_para_df = []
for codigo_municipio, valor in resultados.items():
    lista_para_df.append({'codigo_municipio': codigo_municipio, 'pib_per_capita_2021': valor})

df_pib = pd.DataFrame(lista_para_df)

print("PIB per capita dos 5 primeiros municípios de SC em 2021:")
print(df_pib.head())

3.4. Rede Geodésica (Bases GNSS)

A API de Geodésia permite consultar as estações da Rede Geodésica Brasileira, pontos de referência para levantamentos de alta precisão.

import requests

# URL para buscar todas as estações de Santa Catarina
url_gnss = "https://servicodados.ibge.gov.br/api/v1/geodesia/estacoes?uf=SC"

response = requests.get(url_gnss)
estacoes = response.json()

print(f"Encontradas {len(estacoes)} estações da Rede Geodésica em Santa Catarina.")

# Vamos imprimir os detalhes das 3 primeiras estações
print("\nDetalhes das 3 primeiras estações:")
for estacao in estacoes[:3]:
    print(f"  - Identificador: {estacao['identificador']}")
    print(f"    Nome: {estacao['nome']}")
    print(f"    Coordenadas: Lat {estacao['latitude']}, Lon {estacao['longitude']}")
    print(f"    Situação: {estacao['situacao']}")

4. Laboratório Prático: Mapa Completo de um Município

Vamos juntar tudo em um script final! O objetivo é criar um mapa de um município que mostre sua malha territorial, as estações da rede geodésica dentro ou próximas a ele e um marcador no centroide com o PIB per capita.

import requests
import folium
import pandas as pd

# --- PARÂMETROS ---
NOME_MUNICIPIO = "Joinville"
UF_CODE = "42"  # Santa Catarina

# --- 1. BUSCAR CÓDIGO DO MUNICÍPIO ---
url_municipios = f"https://servicodados.ibge.gov.br/api/v1/localidades/estados/{UF_CODE}/municipios"
response_municipios = requests.get(url_municipios)
municipios_sc = response_municipios.json()
codigo_municipio = None
for municipio in municipios_sc:
    if municipio['nome'] == NOME_MUNICIPIO:
        codigo_municipio = municipio['id']
        break
if not codigo_municipio:
    raise Exception(f"Município '{NOME_MUNICIPIO}' não encontrado.")

print(f"Código IBGE de {NOME_MUNICIPIO}: {codigo_municipio}")

# --- 2. BUSCAR MALHA MUNICIPAL (GEOJSON) ---
url_malha = f"https://servicodados.ibge.gov.br/api/v3/malhas/municipios/{codigo_municipio}"
params = {'formato': 'application/vnd.geo+json'}
response_malha = requests.get(url_malha, params=params)
malha_geojson = response_malha.json()
print("Malha municipal obtida.")

# --- 3. BUSCAR DADO DE PIB PER CAPITA ---
url_pib = f"https://servicodados.ibge.gov.br/api/v3/agregados/5938/periodos/2021/variaveis/37?localidades=N6[{codigo_municipio}]"
response_pib = requests.get(url_pib)
pib_data = response_pib.json()[0]['resultados'][0]['series'][0]['serie']['2021']
print(f"PIB per capita (2021): R$ {pib_data}")

# --- 4. BUSCAR ESTAÇÕES GNSS ---
url_gnss = "https://servicodados.ibge.gov.br/api/v1/geodesia/estacoes?uf=SC"
response_gnss = requests.get(url_gnss)
estacoes_sc = response_gnss.json()
print(f"Encontradas {len(estacoes_sc)} estações GNSS em SC.")

# --- 5. CRIAR O MAPA ---
# Centralizar o mapa no centroide do município
centroide = malha_geojson['features'][0]['properties']['centroide']
mapa_final = folium.Map(location=[centroide[1], centroide[0]], zoom_start=10)

# Adicionar a malha
folium.GeoJson(malha_geojson, name=f"Limite de {NOME_MUNICIPIO}").add_to(mapa_final)

# Adicionar marcador com o PIB
folium.Marker(
    location=[centroide[1], centroide[0]],
    popup=f"<strong>{NOME_MUNICIPIO}</strong><br>PIB per capita (2021): R$ {pib_data}",
    icon=folium.Icon(color='blue', icon='info-sign')
).add_to(mapa_final)

# Adicionar marcadores para as estações GNSS
for estacao in estacoes_sc:
    folium.Marker(
        location=[estacao['latitude'], estacao['longitude']],
        popup=f"<strong>{estacao['identificador']}</strong><br>{estacao['nome']}<br>Situação: {estacao['situacao']}",
        tooltip=estacao['identificador'],
        icon=folium.Icon(color='green', icon='signal', prefix='fa')
    ).add_to(mapa_final)

folium.LayerControl().add_to(mapa_final)
mapa_final.save("mapa_completo_municipio.html")
print("\nMapa 'mapa_completo_municipio.html' salvo com sucesso!")

Esse fluxo integra dados administrativos, espaciais e econômicos em um único produto interativo, pronto para ser compartilhado.
