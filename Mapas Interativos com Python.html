Mapas Interativos com Python e Folium

Transforme dados geográficos em experiências interativas: domine o Folium, personalize visualizações e publique mapas navegáveis que contam histórias.

Você vai aprender a

  • ▹Criar mapas interativos com poucos comandos em Python.
  • ▹Personalizar zoom, marcadores, ícones e camadas de fundo.
  • ▹Construir visualizações temáticas a partir de arquivos CSV.

1. Por que mapas interativos?

Análises geoespaciais ganham impacto quando os dados podem ser vistos no lugar onde acontecem. Mapas interativos oferecem zoom, clique e camadas temáticas — tudo que precisamos para transformar planilhas em narrativas visuais convincentes.

O Folium funciona como uma ponte entre Python e a biblioteca JavaScript Leaflet, permitindo construir experiências web sem sair do ecossistema que você já domina. Vamos entender como instalar, configurar e expandir essa ferramenta.

Instalação rápida

Certifique-se de ter o Folium disponível no ambiente virtual do projeto.

pip install folium

Se trabalhar com dados tabulares, instale também o pandas: pip install pandas.

2. Folium em ação

O que o Folium entrega?

  • Simplicidade: com poucas linhas você gera um HTML interativo completo.
  • Flexibilidade: adicione marcadores, linhas, polígonos e mapas coropléticos.
  • Interatividade: suporte a zoom, popups, tooltips e alternância de camadas.
  • Portabilidade: exporte o resultado como um arquivo único compatível com qualquer navegador.

Do código ao navegador

O fluxo básico envolve criar um objeto folium.Map, adicionar camadas ou marcadores e salvar o resultado.

import folium

# Coordenadas do IFSC - Campus Florianópolis
coordenadas_ifsc = [-27.59397, -48.54235]

mapa = folium.Map(location=coordenadas_ifsc)

# Opção A: gerar arquivo definitivo
mapa.save('meu_primeiro_mapa.html')
print("Mapa salvo com sucesso! Abra 'meu_primeiro_mapa.html'.")

# Opção B: abrir em um arquivo temporário
# mapa.show_in_browser()
# print("Mapa aberto no navegador!")

O arquivo HTML gerado é autossuficiente. Para garantir tiles offline, utilize provedores que ofereçam cache ou tiles locais.

3. Organização do projeto

Dica do professor

Organize scripts, dados CSV e arquivos HTML numa mesma pasta do projeto. Assim você garante caminhos relativos simples e facilita o compartilhamento do material inteiro com a turma.

4. Controlando zoom e layout

Ajustando os parâmetros de zoom

Escolha um nível de zoom que conte a história certa: bairros, cidades ou regiões amplas. Limites de zoom evitam que o usuário perca o contexto.

mapa_com_zoom = folium.Map(
    location=[-27.59397, -48.54235],
    zoom_start=15,
    min_zoom=10,
    max_zoom=18,
    width='80%',
    height='500px'
)

mapa_com_zoom.save('mapa_com_zoom.html')

Experimente valores de zoom_start iguais a 5, 12 e 18 para comparar níveis estadual, municipal e de rua.

5. Trabalhando com marcadores

Adicionando marcadores básicos

Crie o marcador primeiro e depois fixe-o no mapa. Esse padrão facilita loops com dezenas de pontos.

mapa = folium.Map(location=[-27.59397, -48.54235], zoom_start=15)

marcador_ifsc = folium.Marker(
    location=[-27.59397, -48.54235],
    popup='IFSC - Câmpus Florianópolis',
    tooltip='Clique para ver detalhes'
)

marcador_ifsc.add_to(mapa)
mapa.save('mapa_com_marcador.html')

Ícones temáticos

Diferencie categorias com cores e símbolos da biblioteca Font Awesome.

mapa = folium.Map(location=[-27.59, -48.54], zoom_start=14)

folium.Marker(
    location=[-27.59397, -48.54235],
    popup='Local de estudo',
    icon=folium.Icon(color='blue', icon='book', prefix='fa')
).add_to(mapa)

folium.Marker(
    location=[-27.5975, -48.545],
    popup='Área de lazer',
    icon=folium.Icon(color='green', icon='tree', prefix='fa')
).add_to(mapa)

folium.Marker(
    location=[-27.588, -48.54],
    popup='Atenção: histórico de alagamentos',
    icon=folium.Icon(color='red', icon='exclamation-sign', prefix='fa')
).add_to(mapa)

mapa.save('mapa_icones_personalizados.html')

6. Atenção às licenças

Crédito das camadas é obrigatório

Sempre informe a origem dos tiles (atributo attr). Alguns provedores exigem cadastro ou restringem o uso comercial. Leia a licença antes de publicar seu mapa.

7. Alternando camadas base

Configurando mapas de fundo

Adicione múltiplas camadas e permita que o usuário troque entre ruas, satélite ou relevo.

mapa = folium.Map(location=[-27.59, -48.54], zoom_start=13)

folium.TileLayer('OpenStreetMap', name='Mapa de Ruas').add_to(mapa)
folium.TileLayer(
    tiles='https://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}',
    attr='Esri',
    name='Satélite'
).add_to(mapa)
folium.TileLayer('Stamen Terrain', name='Relevo').add_to(mapa)

folium.LayerControl().add_to(mapa)
mapa.save('mapa_com_camadas.html')

8. Laboratório prático

Balneabilidade das praias

Vamos combinar Pandas e Folium para criar um mapa temático com cores por situação de balneabilidade.

Passo 1 · Dados (salve como Pontos.csv)

Praia,Longitude,Latitude,Situação
Praia da Barra,-48.573928,-27.600218,Boa
Praia Brava,-48.574293,-27.600404,Boa
Praia Grande,-48.57454,-27.600603,Ruim
Praia Comprida,-48.574626,-27.600705,Regular
Praia da Joaquina,-48.4490,-27.6325,Boa
Praia Mole,-48.4358,-27.6092,Boa
Praia de Canasvieiras,-48.4116,-27.4322,Ruim

Passo 2 · Script em Python

import folium
import pandas

df_praias = pandas.read_csv('Pontos.csv')
media_lat = df_praias['Latitude'].mean()
media_lon = df_praias['Longitude'].mean()

mapa_praias = folium.Map(location=[media_lat, media_lon], zoom_start=12)
folium.TileLayer(
    'https://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}',
    attr='Esri',
    name='Satélite'
).add_to(mapa_praias)

cores_situacao = {
    'Boa': 'green',
    'Regular': 'orange',
    'Ruim': 'red'
}

for _, praia in df_praias.iterrows():
    situacao = praia['Situação']
    cor_do_icone = cores_situacao.get(situacao, 'gray')

    folium.Marker(
        location=[praia['Latitude'], praia['Longitude']],
        popup=f"{praia['Praia']}
Situação: {situacao}",
        tooltip=praia['Praia'],
        icon=folium.Icon(color=cor_do_icone, icon='tint', prefix='fa')
    ).add_to(mapa_praias)

folium.LayerControl().add_to(mapa_praias)
mapa_praias.save('mapa_balneabilidade.html')

print('Mapa das praias gerado com sucesso!')

Use .get() para garantir que valores inesperados no CSV sejam tratados com uma cor neutra.

9. Explorando plugins e extras

Próximos passos com plugins

Amplie a experiência com extensões já incluídas no Folium.

Marcadores arrastáveis

import folium
from folium.plugins import DragMarker

mapa = folium.Map(location=[-27.59, -48.54], zoom_start=14)

DragMarker(
    [-27.59397, -48.54235],
    popup='Arraste-me para a posição correta!'
).add_to(mapa)

mapa.save('mapa_marcador_arrastavel.html')

Popups ricos

import folium

html_popup = """
    
Ponte Hercílio Luz

    

    
A ponte é um cartão-postal de Florianópolis.

    Saiba mais
"""

iframe = folium.IFrame(html_popup, width=280, height=280)
popup = folium.Popup(iframe, max_width=280)

mapa = folium.Map(location=[-27.5930, -48.5615], zoom_start=16)
folium.Marker(
    location=[-27.5930, -48.5615],
    popup=popup,
    tooltip='Clique para ver!'
).add_to(mapa)

mapa.save('mapa_popup_html.html')

Mapas de calor

import folium
from folium.plugins import HeatMap
import random

mapa = folium.Map(location=[-27.59, -48.54], zoom_start=12)

pontos = [
    [-27.59 + (random.random() - 0.5) * 0.1,
     -48.54 + (random.random() - 0.5) * 0.2]
    for _ in range(100)
]

HeatMap(pontos).add_to(mapa)
mapa.save('mapa_de_calor.html')

10. Curiosidade

Curiosidade: integração com Jupyter

O Folium renderiza mapas diretamente em notebooks Jupyter ou Google Colab. Use mapa como última instrução da célula ou display(mapa) para visualizar sem salvar arquivos. Isso acelera protótipos e relatórios exploratórios.

11. Exercícios rápidos

1. Ajuste de zoom personalizado

Usando o mapa básico, aplique min_zoom e max_zoom para limitar a navegação a uma região de interesse.

2. Marcadores em lote

Crie uma lista de coordenadas e, com um for, adicione marcadores coloridos automaticamente. Use um dicionário para mapear categorias a ícones.

3. Popup multimídia

Monte um popup com HTML, incluindo uma imagem e um link. Salve o resultado como mapa_popup_personalizado.html.

12. Conclusão

Fechamento

Continue combinando Folium com bibliotecas como GeoPandas e Shapely para análises espaciais mais complexas — os mapas interativos serão a vitrine ideal para seus insights.
